LTS is the last kernel declared to have long term support (lts) by Linus Torvalds, currently 6.2. Ie it will be a bit older, does not have latest features but get security patches.

AMD Kernel is similar to Zen, but has different patches, optimized for AMD support. You also have to Google to find what exactly.

For both things, what they offer will also change from kernel version to kernel Version, because stuff is integrated into the mainline kernel (especially true for AMD kernel).

Mainline kernel is the most recently released kernel as is.
