I wasted my 3-4 hours to find the solution for this issue and just wanted to share with people so they can save their time. 
---- SOLUTION ---- 
1st option: 

rm -rf /etc/pacman.d/gnupg

pacman-key --init

pacman-key --populate archlinux blackarch

pacman-key --update --keyserver keyserver.ubuntu.com

2nd option: 
You could try to change to pgp.mit.edu's IP address using the following command: 

curl -O https://blackarch.org/strap.sh


chmod +x strap.sh

sha1sum strap.sh: it should match with the information on downloads


sh strap.sh
Please follow these steps and remove the lock file before running sudo pacman -Syu at the end.
