Garuda does not "prohibit" dual booting. It just is not supported, because windows tends to break Linux. üòÅ
But it works OK, if you follow some rules (for example disable fast boot in windows and always shut it down properly).
That being said, Garuda will need to install its Bootloader into the system efi partition, which will be on your internal drive. There should be sufficient space on there. Also, usually, the system should boot grub (Linux Bootloader) by default, which gives the option to boot windows. Otherwise you'll have to mangle with uefi settings every time you want to boot Linux.

But that's it. Garuda won't change uefi settings.
