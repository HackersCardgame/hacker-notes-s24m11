In my experience water has few advantage. Especially for CPU you get nice big fans and won't profit much from the fans on the radiator in terms of noise (heat transfer can be better). If you connect GPU cooler and Northbridge, too, it get's better but much more complicated. Today IMHO it is better to invest in a good CPU cooler and a GPU with a better cooler (those three fan style designs). Then your computer will be quite nice in terms of noise.

The biggest headache with water cooling is, that it always is a big mess to replace any components... And it is very expensive compared to the other opinions.

I experimented a lot with it, but won't recommend it today (and also not back then). If you want to experiment with it because of interest, sure, go ahead. But realistically, you can achieve the same things buying a bit better fans than stock and a good case.

Nah, Sy can really break your system. y means "refresh package list from repository", so if you install just one package, that one will be the newest version, but all other packages will be the same version already installed. If versions of a library change that way, you can easily end in a state where a lot of programs refuse to even start.

If you omit y, the old version of the package will be installed, if still there or the command will fail, which means you need to update whole system.

If you add u all packages will be updated and the additional package installed in that process.
