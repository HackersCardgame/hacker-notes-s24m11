:reg

You can put anything in these registers using yank or delete/cut commands. To select them you need to use

"<RegName>

Just like text you can also record your actions and store them in register and have vim repeat them for you. Eg,

I have,
for(auto var : foo)

I can press
q
Followed by some register name like,
a

Now my actions will be recorded in the a register.

I made the changes like this,

for(const auto &var : foo)

q

0f( iconst<Esc>wi&<Esc>0:w<CR>

This basically searches for the first ( since the start of the line, enters insert mode writes const then enters normal mode uses w to jump after the auto keyword entered insert mode again typed an & there entered normal mode again. Went to start of line saved my actions.

@a

Or

@@

To use the last register I used.
