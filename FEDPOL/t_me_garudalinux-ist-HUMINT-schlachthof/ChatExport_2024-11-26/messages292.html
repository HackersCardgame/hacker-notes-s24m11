Wayland and X11 basically do the same thing.
X11 is ancient and has some security flaws. It is overloaded with features and many stuff can be achieved in multiple ways. Most software supports X11.
Wayland is new and should replace X11. Not every software can support Wayland natively today, but there is xwayland, which helps with that. Some stuff is more complicated with wayland, because it cares more for security and just is different and not (yet?) so overloaded with features.

TL;DR: try to use wayland. If you run into trouble, try to use x11.
