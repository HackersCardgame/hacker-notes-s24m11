If the drive has an hardware issue, you will lose all data on it or at least all data that is affected by the issue (if the issue is a local one, like hdd platter wearing down).

It does not matter much if the data is encrypted or not. It could be that encrypted data is stored more compact and therefore more data is affected or the decryption needs a certain blocksize to work and therefore some bits that do not fit the blocksize are additionally affected...

You could, of course, have bad luck and loose the encryption key. Usually the key is stored encrypted on the drive and decrypted using the password you enter, so the password is not the key used to encrypt / decrypt the data on the drive. If that happens, all data will be lost, yes. But that is a few bytes : size of harddisk chance of happening. Should be quite small.
