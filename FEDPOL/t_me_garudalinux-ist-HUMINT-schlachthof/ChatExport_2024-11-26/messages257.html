don't worry learning how to use linux has nothing to do with having a computer science degree. Though since you are new here I will tell you the usual protocol.

In Garuda there is a special bash script that automates the solution for most the bugs that the recent updates might bring you. Hence we use garuda-update for updating our systems.

Since garuda is based on arch most of the solutions that apply to arch apply to garuda as well.

you use pacman to install/remove stuff from repos and paru to use the aur.

If you ask for assistance here you might get answers most of the time as well but please try not to do so as we have a forum for this and a lot of the times you might just get ignored since here you message asking for assistance might get lost in the stream of new messages.

Don't treat forum as general technical help group please.

If you are stuck somewhere for instance an app is not working you:
1. first launch the app from terminal (aka type the name of binary in the terminal) to see it's logs and perform the actions that give you grief and search the internet for the error message spit out by the logs.
2. if the you can't find the error on internet try looking at the project's github repo and see if someone else has the same issue as you and look for the solution the community reached or if they don't then create a new issue yourself.
3. if the issue is garuda specific visit the forum at forum.garudalinux.org and create a new issue following the proper template for answers.
