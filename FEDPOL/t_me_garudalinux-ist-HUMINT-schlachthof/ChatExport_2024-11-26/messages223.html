it probably is. With fixed release distros you usually have not trouble. Only with packages that are not in the repository or versions that are not in there.
You might have similar troubles when upgrading from one release to the other (especially with packages not in the repository).

The biggest issue with fixed release distros is that you will have quite old versions of all software. This can be an issue at times. It probably is nasty for gaming (and probably is the reason why lutris and steam just sideload their own wine and dxvk versions).

I tend to an ubuntu system for more than 10 years now. All in all, if you look for more stability, go for a fixed release distro. But you will probably end up with adding newer versions of software here and there and run into problems with them from time to time, too. *shrug*
