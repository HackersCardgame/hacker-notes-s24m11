That question does not make much sense. üòÅ
Lts means it is an older kernel version that gets long term support (lts), so people that don't want to run into trouble with newer stuff in the kernel can stay on it for a few years.
Zen is based on mainline kernel, adds a few patches and has a certain configuration. It usually is a few weeks behind release of mainline kernel.
The same is for what you see as package "Linux", which is the arch default kernel.
There also is Linux-mainline which is what Linus Torvalds is working on. You'll get the newest features there, but also release candidates. That probably is the most kernel most issue prone.

On the other hand the "optimizations" in the other kernels are not necessary stable either or might not work too well for your hardware / use cases.
