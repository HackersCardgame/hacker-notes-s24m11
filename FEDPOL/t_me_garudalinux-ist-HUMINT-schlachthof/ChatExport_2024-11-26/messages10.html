Damian Höster
@Librewish The easiest way to enable automatic GRUB updates after every Timeshift Btrfs snapshot is to
1. Create subvolume timeshift-btrfs on root partition
2. Create folder .snapshots in the root subvolume (@)
3. Add fstab entry like this:
UUID=<UUID> /.snapshots btrfs subvol=timeshift-btrfs 0 0
4. Run sudo systemctl enable grub-btrfs.path and sudo systemctl start grub-btrfs.path in the new system.
